Proyecto Poncho

Base de html y css basada en Bootstrap para la creación de sitios pertenecientes a la Administración Pública Nacional de la República Argentina.

El desarrollo del código y la documentación de la v0.1 de Poncho están en desarrollo. Se aceptan issues con dudas o sugerencias, pero por el momento no vamos a estar aceptando pull requests.

Dependencias

El archivo poncho.css no tiene dependencias directas fuentes ni imágenes.
Para que se vea y funcione bien deben estar los siguientes archivos en la web:

Tipografías

Los archivos se encuentran en las carpetas css y fonts

  • roboto-fontface.css
  • droid-serif.css
Jquery (Versión 1.10+)

  • jquery.js
Bootstrap (Versión 3.3.4+)
  • bootstrap.css
  • bootstrap.js
